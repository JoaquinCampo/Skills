# Color Palette

Use color to encode **meaning**, not decoration. Same layer = same color everywhere.

| Layer | Background | Border | Use for |
|-------|-----------|--------|---------|
| API / Input | `#DBEAFE` | `#3B82F6` | Routers, endpoints, frontend |
| Service / Logic | `#EDE9FE` | `#7C3AED` | Services, business logic |
| Storage / Data | `#D1FAE5` | `#22C55E` | Repositories, database, models |
| External | `#FEF3C7` | `#F59E0B` | Third-party APIs, integrations |
| Error path | `#FFCDD2` | `#EF4444` | Rejection, failure states |
| Success path | `#C8E6C9` | `#22C55E` | Final output, happy path |
| Note/Annotation | `#FFFDE7` | `#FBC02D` | Constraints, invariants |
